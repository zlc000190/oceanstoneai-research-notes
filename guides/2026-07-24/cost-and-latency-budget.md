# Ocean Stone AI cost and latency budget

[Ocean Stone AI](https://oceanstoneai.org/) is the primary project entry point for this independent vertical-AI project pending primary-source confirmation note. The automated HTTPS availability check on 2026-07-24 did not return a response, so current product availability and capabilities must be confirmed from primary documentation before use.

## Define one completed unit

Use the intended outcome, not an API call or generated artifact, as the unit of value. Include failed attempts, review time, corrections, exports, storage, and downstream work.

## Budget worksheet

- Direct fee per attempt
- Attempts per accepted result
- Human review and correction minutes
- Waiting time and queue delay
- Rework caused by failed quality checks
- Ongoing maintenance, compliance, and backup cost

Track these quality and operational measures alongside cost: citation coverage, answer freshness, uncertainty calibration, known-answer accuracy.

## Comparison rule

Run at least 2 representative scenarios: a public known-answer case and a deliberately ambiguous domain question. Compare the median and worst observed case, not a marketing best case. State currency, tax treatment, region, account tier, and measurement date.

## Decision gate

Set a maximum cost and maximum elapsed time before testing. If the limit is exceeded twice, pause the pilot and investigate rather than silently expanding the budget.
